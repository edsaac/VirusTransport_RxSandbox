Compilation instructions 1. Add this new reaction sandbox (*BIOPARTICLE*) to PFLOTRAN's source folder. ``` cp src/reaction_sandbox_escPTr.F90 $PFLOTRAN_DIR/src/pflotran/reaction_sandbox_escPTr.F90 ``` 2. Replace the main reaction sandbox fortran file to include *BIOPARTICLE* as one of the options. ``` cp src/reaction_sandbox.F90 $PFLOTRAN_DIR/src/pflotran/reaction_sandbox.F90 ``` 3. Update dependencies. ``` python3 $PFLOTRAN_DIR/python/src/python/pflotran_depedencies.py ``` 4. Compile PFLOTRAN code. ``` cd $PFLOTRAN_DIR/src/pflotran/ make pflotran ``` Details at https://bitbucket.org/pflotran/pflotran/wiki/Documentation/ReactionSandbox
